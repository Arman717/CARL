=======================
How to Install CARL
=======================

.. role:: bash(code)
    :language: bash

First clone our GitHub repository:

.. code-block:: bash
    #TODO: adapt link
    git clone https://github.com/automl/GenRL.git
    cd clone

We recommend installing within a virtual environment:

.. code-block:: bash

    conda create -n carl python=3.9
    conda activate carl
